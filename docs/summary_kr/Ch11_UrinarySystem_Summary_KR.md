# Chapter 11. 비뇨계통 (Urinary System)

## 1. 비뇨계통의 구조

### 비뇨계통의 개요
- **구성**: 콩팥(신장, kidney), 요관(ureter), 요도(urethra), 방광(urinary bladder)
- **기능**:
  - 혈액 내의 노폐물을 소변으로 배출
  - 소변 배출량을 조절하여 혈압 조절
  - 전해질의 평형 유지
  - 수소이온(H⁺) 배설을 조절하여 산-염기 평형 조절
  - 레닌(renin)을 방출하여 혈압 조절
  - 에리트로포이에틴(erythropoietin)을 분비하여 적혈구 생성 조절

### 레닌 (Renin)
- **정의**: 일종의 단백질분해효소로 들세동맥벽의 토리옆세포(사구체옆세포, JGA)에서 분비
- **작용기전**: 혈장의 α2-globulin 분해 → 안지오텐신Ⅰ(아미노산 10개 펩티드) → 가수분해 → 안지오텐신Ⅱ → 말초혈관 수축 → 혈압 상승
- **안지오텐신Ⅱ의 혈압 상승작용**: 노르아드레날린의 40~50배
- **분비 촉진**: 콩팥에 유입되는 동맥혈압이 저하될 때

### 콩팥의 외부구조
- **위치**: 배안 뒤벽의 위쪽, 척주의 양쪽에 있는 한 쌍의 복막뒤기관(retroperitoneal organ)
- **높이**: 위끝은 열두째등뼈, 아래끝은 셋째허리뼈
- **좌우 차이**: 오른쪽 콩팥은 왼쪽 콩팥보다 약간 낮은 위치
- **형태**: 강낭콩 모양의 실질장기, 길이 약 10cm, 무게 약 130g
- **콩팥문(신문, hilum)**: 안쪽 중앙부의 함몰한 부위, 혈관·림프관·신경·요관 출입

### 콩팥의 내부구조
- **겉질(피질, cortex)**: 바깥층, 적갈색
- **속질(수질, medulla)**: 속층, 적색
- **콩팥기둥(신주, renal columns)**: 겉질이 파고 들어가서 생긴 구조
- **콩팥피라미드(신추체, renal pyramids)**: 콩팥기둥에 의해 10여 개로 나뉨
- **콩팥유두(신유두, renal papilla)** → **콩팥잔(신배, renal calices)** → **콩팥깔때기(신우, renal pelvis)** → 요관

### 콩팥단위 (Nephron)
- **정의**: 콩팥실질의 구조물, 한쪽 콩팥에 약 100만 개
- **구성**: 콩팥소체(신소체, renal corpuscle) + 콩팥세관(세뇨관, renal tubule)
- **종류**: 겉질콩팥단위(cortical nephron), 속질콩팥단위(juxtamedullary nephron)

### 콩팥소체 (Renal Corpuscle)
- **위치**: 겉질 속, 직경 약 0.2mm
- **구성**: 토리(사구체, glomerulus) + 토리주머니(보우만낭, Bowman's capsule)
- **토리**: 들세동맥(수입세동맥, afferent arteriole) → 모세혈관이 뭉쳐 공 모양 → 날세동맥(수출소동맥, efferent arteriole)
- **토리주머니**: 안팎 2엽의 상피막, 관내강에 여과액(filtrate)이 고임

### 콩팥세관의 경로
- 토리주머니 → **토리쪽곱슬세관(근위세뇨관, proximal convoluted tubule)** → **콩팥세관고리(Henle's loop)** → **먼쪽곱슬세관(원위세뇨관, distal convoluted tubule)** → **집합세관(collecting duct)** → 콩팥유두 → 콩팥잔 → 콩팥깔때기

### 콩팥의 혈관
- **혈류량**: 양쪽 콩팥의 1분간 혈액유통량은 1,000~1,200mL로 심장 박출량의 약 1/4
- **경로**: 콩팥동맥 → 엽사이동맥 → 활꼴동맥(궁상동맥) → 소엽사이동맥 → 들세동맥 → 토리 → 날세동맥 → 세관주위 모세혈관그물 → 소엽사이정맥 → 활꼴정맥 → 엽사이정맥 → 콩팥정맥
- **곧은동맥(직세동맥, rectal arteriole)**: 날세동맥의 가지, 콩팥세관고리와 집합관 주변에 분포

## 2. 요관, 방광, 요도

### 요관 (Ureter)
- **길이**: 25~28cm의 가느다란 관
- **구조**: 점막, 근육층, 바깥막
- **근육층**: 속층=세로근육, 바깥층=돌림근육(소화관과 반대), 아랫부분에는 바깥에 또 한 층의 세로근육층
- **꿈틀운동**: 1분에 2~4회

### 방광 (Urinary Bladder)
- **위치**: 골반안의 두덩결합의 뒷면
- **용적**: 500~700mL
- **방광삼각(vesical trigone)**: 왼·오른쪽 요관구멍과 속요도구멍 3개를 연결하는 부분
- **벽 구조**: 이행상피 점막 + 근육층(속세로근·중간돌림근·바깥세로근의 3층) + 바깥막
- **배뇨근(detrusor muscle)**: 방광 근육층의 총칭

### 요도 (Urethra)
- **남성요도**: 20cm, 오줌 배출 + 정액 배출로 기능
- **여성요도**: 3~4cm, 바깥요도구멍은 질구멍의 앞·음핵의 뒤에 위치
- **여성 감염 위험**: 요도가 짧고 항문에 가까워 대장균 이동이 용이 → 요도염 → 방광염 → 콩팥깔때기콩팥염(신우신장염)

## 3. 비뇨기의 생리

### 오줌의 생성 과정
- **3가지 과정**: 토리여과(glomerular filtration) → 세관재흡수(tubular reabsorption) → 세관분비(tubular secretion)

### 오줌 (Urine)
- **pH**: 대략 5.0~7.0 (육식=산성, 야채=알칼리성)
- **비중**: 1.015~1.025, 오줌양에 반비례
- **1일 오줌양**: 1~1.8L
- **색깔**: 우로크롬(urochrome)
- **고형성분(하루 약 60g)**: 무기물 25g(NaCl 60%) + 유기물 35g
- **유기물**: 요소(urea)가 가장 많아 질소량의 90%, 나머지 10%=요산, 크레아티닌, 암모니아, 히푸르산, 아미노산

### 토리여과 (Glomerular Filtration)
- **원리**: 혈압에 의한 여과, 토리주머니 내에 단백질 외 모든 성분 여과
- **여과압 계산**: 혈압(70mmHg) - 혈장아교질삼투압(25mmHg) - 보우만공간내압(10mmHg) = **35mmHg**
- **토리여과속도(GFR)**: 안정 시 어른남자 평균 125mL/min
- **콩팥혈장 유통량**: 평균 660mL/min
- **콩팥 혈류량**: 1,200mL/min

### 세관 재흡수 (Tubular Reabsorption)
- **재흡수율**: 여과액의 99% 이상
- **주요 부위**: 대부분 토리쪽곱슬세관(근위세뇨관)에서 곧은세관 사이
- **재흡수 물질**: 수분, 포도당, 아미노산, Na⁺, Cl⁻ 등
- **호르몬 조절(먼쪽곱슬세관)**:
  - **항이뇨호르몬(ADH)**: 수분 재흡수 촉진
  - **알도스테론(aldosterone)**: Na⁺ 재흡수 촉진, K⁺ 배설 증가, HCO₃⁻ 재흡수 → 오줌 산성화

### 세관 분비 (Tubular Secretion)
- **분비 물질**: 크레아티닌, 암모니아, 히푸르산, K⁺, HPO₄⁻
- **이물질 배설**: 파라아미노히푸르산(PAH), 페놀레드, 페니실린 → 재흡수되지 않고 적극 분비

### 청소율 (Clearance)
- **정의**: 어떤 물질의 1분간 오줌 내 배설량이 얼마만큼의 혈장 중에 포함되어 있는지를 표시
- **공식**: C(mL/min) = UV/P
- **이눌린(inulin), 만니톨(mannitol)**: 토리에서 여과되나 세관에서 재흡수·분비 없음 → 청소율 = 토리여과속도

### 배뇨 (Urination)
- **요의 발생**: 방광에 200~300mL 저류 시 내압 급상승 → 요의(배뇨감) 발생
- **배뇨 시 방광 내 압력**: 30cmH₂O
- **배뇨반사**: 일차적 중추가 엉치척수에 있는 자율신경성 반사
- **신경 지배**: 교감신경은 무관, 부교감신경은 촉진
- **고위중추**: 뇌에서 배뇨반사를 의식적으로 정지 또는 유발 가능

### 산염기평형의 조절
- **체액 pH**: 7.35~7.45
- **산성증(acidosis)**: pH가 정상값보다 낮은 상태
- **알칼리증(alkalosis)**: pH가 정상값보다 높은 상태
- **pH완충계(buffer system)**: 중탄산이온, 인산염, 단백질의 3단계
- **콩팥의 역할**: H⁺를 Na⁺과 교환하여 오줌에 배설, 암모니아(NH₃) 생성

### 체액량의 삼투압 조절
- **불감성 증발(insensible evaporation)**: 피부면에서 하루 800mL
- **불가피뇨(obligatory urine)**: 500mL
- **조절 가능한 오줌양**: 900mL
- **수분 과잉** → 부종(edema), **수분 부족** → 탈수(dehydration)
- **탈수 시**: 삼투압 상승 → 갈증 → 수분섭취 증가 + 항이뇨호르몬 분비 증가 → 농축된 오줌 배설
- **삼투압 저하 시**: 항이뇨호르몬 분비 억제
- **Na⁺ 조절**: 알도스테론(aldosterone) 분비 관여
