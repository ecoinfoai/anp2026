# 제12장 생식 및 발생 (Reproduction and Development)

## 남성생식 (Male Reproduction)

---

### 구조

#### 고환 (Testis)
- **위치**: 음낭(scrotum) 내
- **크기**: 무게 약 8g의 편평타원형 기관
- **발생**: 태아시기에 콩팥 앞면 아래에서 샅굴(서혜관)을 통해 음낭 내로 하강
- **내부구조**:
  - **소엽**: 결합조직성 막으로 칸막이된 약 100개의 소엽으로 구성
  - **정세관** (seminiferous tubule) → **고환수출관** (약 10개) → **부고환** → **정관** → **사정관** → **요도**
  - **세포**: 정세관 = 정상피 + 지지세포
  - **사이질세포** (간질세포, interstitial cell; Leydig's cell) → 남성호르몬 분비

#### 정로 (Seminal Tract)
정자의 통로 → 부고환 → 정관 → 사정관 → 요도
- **부고환** (epididymis): 10개의 고환수출관을 수집하여 정관으로 연결
- **정관** (vas deferens): 고환의 혈관, 신경과 합쳐지는 정삭(spermatic cord) 형성 → 샅굴 통과 → 골반 내 사정관에 연결
- **사정관** (ejaculatory duct): 정관 + 정낭에서 연결되는 가는 관 → 전립샘 통과 → 요도 개구
- **요도** (urethra): 오줌과 정액(semen) 배출

#### 부속생식샘
- **정낭** (seminal vesicle):
  - 방광 뒷면의 정관팽대부 바깥쪽
  - 정자 저장 + 알칼리성 분비물 분비 (정액의 일부)
  - 기능: 정자 운동 촉진

- **전립샘** (prostate gland):
  - 방광 밑에 위치, 사정관과 요도 시작부를 둘러쌈
  - 밤알 크기
  - 분비: 밤꽃 냄새의 우윳빛 액체
  - 정액 구성: 약 30%
  - 기능: 정자 활성화 + 운동성 유지

- **망울요도샘** (요도구선):
  - 여성의 큰질어귀샘(Bartholin's gland)에 해당
  - 점도 높은 무색 점액성 분비물 분비

#### 바깥음부 (External Genital Organ)
- **음낭** (scrotum): 고환 + 부고환 수용
- **음경** (penis):
  - 요도해면체(corpus cavernosum) + 피부
  - **귀두** (glans penis): 음경 끝부분
  - **발기** (erection): 해면체의 혈관 풍부 → 충혈시 팽대

---

### 기능

#### 정자형성 (Spermatogenesis)
- **배아 초기**: 생식세포 → 고환으로 이동 → **정조세포** (정원세포, spermatogonium)
- **정조세포**: 감수분열 → 4개의 반수체 배우자 생성 (2배체 세포)
- **프로세스**:
  - 정조세포 → (체세포분열) → **1차 정모세포** (primary spermatocyte)
  - 1차 정모세포 → (제1감수분열) → **2차 정모세포** (secondary spermatocyte) 2개
  - 2차 정모세포 → (제2감수분열) → **정자세포** (spermatid) 4개
  - 정자세포 → (정자변성, metamorphosis) → **정자** (spermatoza)
  - 변성 과정: 사이질세포(세르톨리세포)의 작용 필요
- **수정란의 성별**: 정자에 의해 결정
  - X 정자 + 난자(X) → XX (여성, female)
  - Y 정자 + 난자(X) → XY (남성, male)

#### 정자 (Sperm)
- **형태**: 머리 + 중간부 + 꼬리
- **길이**: 50~60 ㎛
- **구조**:
  - **머리**: 핵 + 23개의 염색체 함유 염색질 → **첨단체** (acrosome)에 리소좀 같은 효소 (난세포 침입용)
  - **중간부**: 필라멘트 속 → 사립체들의 나선형 배열 (ATP 생성)
  - **꼬리**: 운동 담당 (많은 ATP 존재)
- **성숙**: 부고환에서 → 운동성 + 다른 변화 획득
- **에너지원**: 정낭의 과당(fructose)
- **특징**: 태생기 이후 전 생애 계속 증식 → 정조세포 수 감소 없음 → 생산능력 유지

#### 발기 (Erection)
- **자극**: 음경 외피 자극 + 성적 욕구
- **신경** → 척수의 엉치척수(천부, sacral portion) 발기중추 흥분 → **부교감신경** 활성
- **결과**: 발기조직의 동맥 확장 + 정맥 수축 → 혈액 충만 → 부피/길이 증가

#### 사정 (Ejaculation)
- **자극**: 성적 흥분 → 척수의 엉치척수 중추 흥분 → **교감신경** 활성
- **기전**: 신경흥분 전도 → 정관 및 부속생식샘의 율동적 경련 → 꿈틀운동(연동, peristalsis)
- **경로**: 고환 → 부고환 → 정관 → 정낭 → 전립샘 → 음경
- **정액** (semen): 정자 + 부속생식샘 분비물의 혼합물
- **배출량**: 약 3 ㎖/회
- **정자 농도**: 약 1억 개/㎖ → 약 3억 개/회 배출
- **기전**: 요도 수축 → 정액 사출 + 전립샘 조임근 수축 → 방광입구 폐쇄

#### 남성의 성반응주기 (Sexual Response Cycle)
- **흥분기** (excitement phase): 발기 + 성감 고조
- **고조기** (plateau phase): 심박수, 혈압, 호흡수, 근육긴장도 강화
- **극치기** (orgasmic phase): 사정 + 육체적 안락감 최고정점
- **해소기** (resolution phase): 생식기 + 신체 반응 + 성욕구 안정화

### 내분비성 조절

#### 남성 성기능 호르몬 조절
- **사춘기** (puberty): 출생 후 10~14세 무렵 → 뇌하수체 앞엽에서 **난포자극호르몬** (FSH) + **황체형성호르몬** (LH) 분비 증대 → 고환 발육 촉진

#### 주요 호르몬
| 호르몬 | 출처 | 기능 |
|--------|------|------|
| **테스토스테론** | 고환의 사이질세포 | • 생식기관 성장<br/>• 남성의 성적 행동 자극<br/>• 난포자극호르몬 존재 시 정자 형성 자극<br/>• 2차성징: 후두/성대 확장, 체모 증가, 근육/뼈 성장 촉진, 피부 두꺼워짐 |
| **생식샘분비자극호르몬** (GnRH) | 시상하부 | • 뇌하수체 앞엽의 LH + FSH 분비 자극 |
| **황체호르몬** (LH) | 뇌하수체 앞엽 | • 고환의 사이질세포 발육 향상 |
| **난포자극호르몬** (FSH) | 뇌하수체 앞엽 | • 정세관의 종자상피 테스토스테론 반응 준비 |

## 여성생식 (Female Reproduction)

---

### 구조

#### 난소 (Ovary)
- **발생**: 태생 2개월경 콩팥 근처에서 → 대골반 → 소골반 가쪽 벽으로 하강 (옆으로 누움)
- **크기**: 3 × 1.5 × 1.0 ㎝ (엄지손가락 크기의 얇은 타원형)
- **무게**: 한쪽 약 7g
- **내부구조**:
  - **겉질** (cortex): 표층 → 여러 발달단계의 **난포** (follicle) 함유
  - **속질** (medulla)

#### 난포 발생
- **원시난포** (primary follicle): 태생기 한쪽 난소에 약 20만 개 이상
- **발생 과정**: 원시난포 → (연령 증가) → 사춘기 약 400개 감소
- **배란주기**: 약 4주마다 1개씩 성숙
- **성숙난포** (mature follicle; 그라프난포, Graafian follicle):
  - 난소 표면 가까이로 이동 → 돌출 → 파열
  - **배란** (ovulation): 성숙된 난자(ovum) 자궁관(배) 내 방출
  - 양쪽 난소에서 교대로 발생

- **난자**: 직경 약 0.2mm의 큰 세포
- **황체** (corpus luteum):
  - 배란 후 난포의 혈출 → 일시적 적색 → **적체** (corpus rubrum) → 황색 변화
  - **황체세포** (lutein cell): 황색 색소 함유
  - **프로게스테론** 분비

#### 자궁관 (Fallopian Tube; 난관, Uterine Tube)
- **길이**: 약 10 ㎝의 가는 관
- **난소쪽**: 깔때기(누두) 모양 확대 → 배안에 개구
- **자궁관술** (난관채, fimbriae of uterine tube): 깔때기 부분 가장자리의 연한 다수의 조직 → 부챗살 모양 돌출
- **운동**: 내부 섬모(cilia) → 자궁 방향 액체 이동
- **난자 수송**: 배란 후 3~7일 소요

#### 자궁 (Uterus)
- **위치**: 방광과 곧창자 사이
- **외형**:
  - **자궁바닥** (저, fundus)
  - **자궁몸통** (체, corpus)
  - **자궁목** (경, cervix) → **질부분** (portio vaginalis): 질 내 돌출부

- **내부**: **자궁강** (자궁공간, uterine cavity) → 이등변삼각형 모양
- **자궁벽**:
  - **자궁속막** (점막, tunica intima; endometrium):
    - 샘상피 + 혈관/샘 풍부
    - 난소호르몬의 주기적 변화 → **자궁속막 주기적 변환**
  - **근육층** (myometrium): 두꺼운 민무늬근육
  - **바깥막** (tunica externa): 장막(serosa)

#### 질 (Vagina)
- **길이**: 약 7 ㎝의 속빈기관
- **위치**: 자궁 → 요도와 곧창자(직장) 사이
- **구조**:
  - **질입구**: 중앙의 **처녀막** (hymen; 결합조직성)
  - **점막**: 중층편평상피 → 가로 주름 다수
  - **기능**: 여성교접기(성교) + 산도(출산길)

#### 바깥음부 (외음부, 음문, Vulva)
**구성요소**:
- **대음순** (labia majora)
- **소음순** (labia minora)
- **음핵** (clitoris): 남성의 음경에 해당 → 발기조직 + 풍부한 감각신경
- **큰질어귀샘** (대전정선, greater vestibular gland; Bartholin's gland)
- **질어귀** (질전정, vestibulum)
- **처녀막** (hymen)

#### 젖샘 (유선, Mammary Gland)
- **구조**: 복합관상꽈리샘 (아포크린땀샘과 유사)
- **남성**: 퇴화 (도관만 남음)
- **여성 발육**:
  - 난포자극호르몬 + 황체형성호르몬 영향 → 사춘기 직전부터 급속 발육
  - 20세 정도: 최고조
  - 40세 이후: 퇴행 시작
  - 갱년기 이후: 위축

- **내부구조**:
  - 주변부: 지방조직
  - 15~20개의 **젖샘** (유선; glandular lobes)
  - 각 샘엽: 다수의 **샘방** + 1개의 **젖샘관** (lactiferous duct)
  - 샘방: 근육상피세포(myoepithelial cell)에 의해 싸임
  - **젖꼭지** (유두, nipple): 젖샘관 개구 부위
  - **젖샘관팽대** (lactiferous sinus): 젖꼭지 밑에서의 확장
  - **젖꽃판** (유륜, areola): 유두 주위 색소 부위
  - **젖꽃판샘** (유륜선, areolar gland): 땀샘과 젖샘의 중간 구조

### 난자발생 (Oogenesis)

- **태아 초기**: 생식세포 → 난소로 이동 → 증식 → 임신 5개월경 약 6~7백만 개의 **난조세포** (난원세포, oogonia)
- **태아기**: 대부분의 난조세포 → 세포소멸(apoptosis)로 사망 (태어나기 전)
- **임신 말기**: 난조세포 → **감수분열 시작** → **1차 난모세포** (primary oocyte)
  - 1차 난모세포: 아직 2배체 (제1감수분열 전기에서 정지)
- **신생아**: 약 2백만 개의 난모세포 보유
- **사춘기**: 약 40만 개로 감소
- **성인 생식기간**: 약 400개만 배란, 나머지 세포소멸

#### 난포 발생 상세
1. **원시난포** (primary follicle)
2. **1차 난포** → 난모세포 + 난포세포 분열 → **과립막세포** (다층)
3. **2차 난포** (secondary follicle) → 소포 형성 → 난포공간 형성
4. **성숙난포** (그라프난포, Graafian follicle)

#### 난모세포의 감수분열
- **1차 난모세포** → (제1감수분열) → **2차 난모세포** + **1개의 극체**
- 2차 난모세포가 세포질 대부분 보유 (극체는 세포질 최소)
- **2차 난모세포** → (제2감수분열) → **난자** (ovum) + **극체** (분열 가능)
- 1차감수분열 극체도 분열 → **2개의 2차극체**
- **최종 결과**: 3개의 극체 + 1개의 난자 (극체 모두 퇴화)

#### 난자 주변 구조
- **부챗살관** (방사관, corona radiata): 난모세포를 둘러싼 과립막세포의 고리
- **투명층** (zona pellucida): 단백질 + 다당류의 겔층 → 정자 침입 저지

#### 황체의 변화
- **황체 수명**: 14 ± 2일
- **다음 월경 전**: 퇴화 시작 → **백체** (corpus albicans)
- **수정 + 착상 시**: 위축 안 됨 → **임신 황체** → 임신 유지에 관여

### 성행동

#### 발기와 윤활작용
- **부위**: 질구멍 주위의 발기조직
- **자극**: 자극 + 중추성 성적 흥분 → 척수의 엉치척수 신경중추 흥분
- **신경**: **부교감신경섬유** → 발기조직에 신경흥분 전도
- **결과**: 바깥음부 부풀어 오름 + 질구멍 좁아짐
- **윤활작용**: 부교감신경 흥분 → **큰질어귀샘** (Bartholin's gland) → 점액 분비 → 성교 시 윤활

#### 성감의 극치감
- **중심부위**: **음핵** (clitoris) → 마찰에 의해 성감 증대
- **극치감** (orgasm): 자궁 + 자궁관 + 주변 조직에서의 율동적 근 경련
- **결과**: 질 내 꿈틀운동(연동) 발생 → 정액을 자궁으로 흡인 용이

### 내분비성 조절

#### 성주기 (Sexual Cycle)
- **정의**: 난소와 자궁의 일정한 주기로의 특유한 변화 반복
- **조절**: 여러 호르몬의 협동작용

#### 호르몬의 역할
| 호르몬 | 출처 | 기능 |
|--------|------|------|
| **에스트로겐** | 난소의 난포 | • 생식기관 성장 + 자궁속막 재형성<br/>• 여성의 2차성징 발달 자극<br/>• 유방, 골반, 엉덩이 지방 침착<br/>• 근육/뼈 성장 증가 → 잘룩한 허리 + 넓은 골반<br/>• 유방의 샘/관 발육<br/>• 두덩털(음모, pubic hair) 발생<br/>• 뼈에 대해: 사춘기 직후 증식 촉진 (4~5년) → 여아의 빠른 키 발육 후 조기 정지 |
| **프로게스테론** | 난소의 황체 + 임신 중 태반 | • 자궁속막의 재형성 향상<br/>• 임신기간 자궁 유지<br/>• 유방의 젖 분비 준비<br/>• 자궁속막 분비기로 변환<br/>• 착상된 수정란 발육에 적합한 조건 준비<br/>• 젖(유즙) 분비 시작<br/>• 자궁 수축 억제 → 유산(abortion) 방지 |
| **황체자극호르몬** (LTH) | 뇌하수체 앞엽 | • 황체에서 프로게스테론 분비 촉진<br/>• 젖 분비에 관여 |
| **황체형성호르몬** (LH) | 뇌하수체 앞엽 | • 1차 난포 → 2차 난포 발육 향상<br/>• 고농도: 배란 촉진 |
| **난포자극호르몬** (FSH) | 뇌하수체 앞엽 | • 1차 난포 → 2차 난포 발육 자극 |

#### 자궁속막의 주기적 변환

**4기 구분**:

1. **월경기** (menstrual phase): 3~5일
   - 원인: 난자 수정 실패 → 황체 위축 → 백체 → 프로게스테론 + 에스트로겐 분비 정지
   - 결과: 호르몬 급감소 → 자궁점막 탈락 → 혈액과 함께 체외 배출 → **월경** (menstruation)

2. **증식기** (proliferative phase)
   - 원인: **에스트로겐** 영향
   - 결과: 자궁점막 증식 → 두꺼워짐

3. **분비기** (secretory phase)
   - 원인: **프로게스테론** 영향 (배란 후 형성 황체로부터)
   - 결과: 자궁점막 충혈 + 분비물 증대
   - 특징: 자궁샘 분비물 영양물질 풍부, 지방질/당원질 증가, 혈류량 증가
   - 의의: 착상될 수정란에 대비

4. **월경전기** (premenstrual phase): 2~3일
   - 자궁속막에 국소빈혈 → 혈관 파열 → 출혈
   - (다음 월경 시작 전)

#### 수정 가능 기간 (Period of Fertility)
- **난자 생존**: 배란 후 **8~24시간** 이내만 정자에 의해 수정 가능
- **정자 생존**: 여성생식기 내 **24~72시간** 동안 생존
- **성행위 시기**: 배란 몇 시간 전 ~ 배란 후 몇 시간까지

### 젖

#### 젖의 배출 (젖 분비 기전)
- **초기**: 유아가 젖꼭지(유두) 빨아도 처음 약 45초간 젖 미배출
- **반사기전**: 호르몬 분비가 관여하는 신경내분비반사

**프로세스**:
1. 유아의 흡인동작 → 젖꼭지에서 감각흥분 시작
2. 흥분 → 시상하부 전도
3. **신경뇌하수체** → **옥시토신** 배출
4. 옥시토신 + 혈류 → 유방의 샘꽈리(선포, alveoli) 벽의 근육상피세포(myoepithelial cells) 수축
5. 결과: 샘세포의 젖(유즙) 배출

**프로락틴**(prolactin):
- 뇌하수체뒤엽 분비
- 기능: 젖 생산 촉진
- 심리적 영향: 공포/불안정 → 젖 배출 지연/정지 → 프로락틴 분비 정지 → 젖 생산 정지

#### 젖의 성분
**구성**:
- **단백질**: 카세인(casein) + 락트알부민(lactalbumin)
- **당류**: 젖당(유당, lactose) = 포도당 + 갈락토오스 (이탄당)
- **지질**: 중성지방 + 콜레스테롤 + 인지질 풍부
- **기타**: 소량의 비타민, 인산칼슘 다량 (철분은 거의 없음)

**영양가 비교**:
| 성분 | 모유 (%) | 우유 (%) |
|------|---------|---------|
| 물 | 88.5 | 87.0 |
| 지방 | 3.3 | 3.5 |
| 락토오스(젖당) | 6.8 | 4.8 |
| 카세인 | 0.9 | 2.7 |
| 락트알부민 + 기타 단백질 | 0.4 | 0.7 |
| 무기질 | 0.2 | 0.7 |

**철분**:
- 태아시기: 모체로부터 철분 수취 → 신생아 간에 많은 양 저장
- 출생 후 약 2개월: 적혈구 생산에 필요한 철분 충분
- 2개월 이후: 유아에게 적절한 철분 섭취 필요 (부족 시 빈혈 위험)

**모유 수유 모체의 영양**:
- 임신 중과 동등한 높은 영양소 섭취 필요
- 특히 단백질, 지질, 칼슘 다량 손실 → 보충 중요

## 인체의 발생과 발육 (Human Development)

---

### 발생 (Development)

#### 수정과 초기 발생
- **수정** (fertilization): 성숙한 난자(1개) + 정자(1개) → **수정란** (fertilized ovum) = 1개의 세포
- **수정 위치**: 대개 자궁관팽대부(난관팽대부)에서 발생

#### 초기 세포 분열
1. **난할** (cleavage): 수정란 → 유사분열 반복
   - 2세포기 → 4세포기 → 8세포기 → 전 배아기
   - 자궁관을 따라 자궁으로 이동

2. **오디배기** (상실기, morula stage):
   - 세포들로 꽉 찬 공 모양
   - 약 4일째 자궁 내 진입

3. **주머니배** (포배, blastocyst):
   - 오디배 → 더 분열 → 내부에 관내강 생성
   - 약 6일경: 속이 비고 속공간이 액체로 채워진 공 모양

#### 착상 (Implantation)
- **시기**: 수정 후 1주일 → 자궁공간 도달 → 착상 시작
- **완성**: 제2주 → 자궁점막에 완전 매몰
- **구조**: 주머니배의 속세포덩이(inner cell mass) → 1차 배엽층 형성

#### 배엽 발생 (Germ Layer Formation)

**태생 제2주**:
- 주머니배의 가쪽면 세포층 → 태반 형성
- 주머니배의 일부 함몰 → 2중 구조 **창자배** (원장배, gastrula):
  - **외배엽** (ectoderm)
  - **내배엽** (endoderm)

**제3주**:
- **중배엽** (mesoderm) 발생 → 외배엽과 내배엽 사이 생성
- 각 배엽의 증식과 분화 → 여러 조직 및 기관 형성

#### 각 배엽에서 발생하는 조직과 기관

- **내배엽** (endoderm):
  - 소화관 상피 + 부속샘
  - 호흡기의 점막상피

- **중배엽** (mesoderm):
  - 뼈대, 근육, 결합조직
  - 혈액, 순환기
  - 비뇨생식기
  - 민무늬근육(평활근) + 결합조직 → 인체 곳곳 분포 → 외배엽성/내배엽성 조직과 결합 → 모든 기관 형성에 관여

- **외배엽** (ectoderm):
  - 표피 + 부속샘 (털, 손·발톱, 피부샘: 땀샘, 피지샘, 젖샘)
  - 신경계통 (뇌, 척수, 말초신경)
  - 감각기의 주요 부분

#### 태아의 보호와 영양
- **양막** (amnion): 태아를 싸는 막
- **양막공간**: 양수(amniotic fluid) 함유 → 태아 부부
- **탯줄** (제대, umbilical cord): 태아 ↔ 모체 연결
- **태반** (placenta): 태아 ↔ 모체 연결 및 물질 교환

### 월령별 태아의 발육

**임신 기간 계산**:
- **기준**: 마지막 월경의 첫날부터
- **출산 시점**: 대개 280일경
- **월령**: 1개월 = 4주 = 28일
- **임신기간**: 280일 = 10개월
- **수정 시점**: 월경 첫날부터 약 2주 후 (계산 시 2주일 빼기)

#### 각 월령별 발육

**제1주**:
- 수정 후 1주일 → 주머니배(포배) 초기상태
- 자궁공간 도달 → 착상 개시

**제2주**:
- 양막공간 + 난황주머니 형성
- 착상 완료

**제3주**:
- 중배엽 발생

**제4주** (월경태령 1개월 말):
- 각 배엽의 상당한 분화 → 각종 기관의 기본구조 형성
- 외배엽: 태아 거의 모든 표면 덮음
- 태아의 외형: 점차 갖춤
- **몸길이**: 약 3~4 ㎜

**제8주 말** (월경태령 2개월 말):
- 사람의 태아로서 고유한 형태 나타남
- 여기까지: **배아** (embryo)
- 이후: **태아** (fetus)
- **몸길이**: 13~15 ㎜

**제12주 말** (3개월 말):
- 바깥음부 형성 → 남녀 구별 용이
- **몸길이**: 50~55 ㎜

**제16주 말** (4개월 말):
- 코와 입 개구
- **몸길이**: 약 10 ㎝

**제20주** (5개월):
- 온몸에 솜털(lanugo) 발생
- 모체: 태아의 움직임(태동) 감지

**제24주** (6개월):
- 피부밑지방(피하지방) 형성

**제28주** (7개월):
- 머리카락 발생
- 눈꺼풀틈새(안열) 갈라짐

**제32주** (8개월):
- 피부밑지방 다량 증가 → 몸 표면 주름 감소
- 몸 표면: 태지(피지샘 분비물 + 박리된 표피)로 덮임
- **특징**: 특별한 보육장치 없어도 모체 밖에서 발육 가능

**제36주** (9개월):
- 머리카락 뚜렷해짐
- 솜털 퇴화 시작

**제40주 말** (10개월; 만삭):
- **몸길이** (마루 부위부터 발바닥까지): 약 50 ㎝
- **몸무게**: 약 3 ㎏

**제27~38주** (3분기; 발생 7~9개월):
- 기본적으로 몸무게 증가
- 허파의 충분한 발달
- 눈꺼풀이 열림
- 신경계통의 추가 발달
- (신경계통은 분만 후에도 계속 성숙)
- **평균 체중**: 약 3.4 ㎏
- **길이**: 50 ㎝

### 태반 (Placenta)

#### 태반의 형상
- **크기**:
  - 직경: 약 20 ㎝
  - 두께: 2.5 ㎝
  - 무게: 450g

- **구조**:
  - **배아 가쪽면**: 편평 → 크고 많은 혈관이 탯줄 쪽으로 모임
  - **모체 가쪽**: 붉고 울퉁불퉁함

#### 탯줄 (Umbilical Cord)
- **구성**:
  - **배꼽동맥** (제동맥, umbilical artery) 2개: 태아 혈액 → 태반
  - **배꼽정맥** (제정맥, umbilical vein) 1개: 태반 → 태아 혈액 운반
  - **길다랗게 변한 난황주머니** (난황낭)
  - **요막관** (allantoic duct): 내배엽으로 형성
  - **배바깥 중배엽**: 위의 구조를 묶는 끈
  - **표면**: 양막상피로 덮임 → 확대된 양막공간 통과

#### 태반의 기능

1. **물질 교환** (영양 공급):
   - 탄수화물, 단백질, 지방, 수분, 무기염류, 비타민, 호르몬, 항체 → 모체 혈액 → 태아 혈액

2. **호흡작용**:
   - O₂: 모체 혈액 → 태아 혈액 (확산)
   - CO₂: 태아 혈액 → 모체 혈액 (확산)

3. **배설**:
   - 태아의 대사에서 생성된 수용성 노폐물질 → 태아 혈액 → 모체 혈액 수송

4. **장벽 기능**:
   - 세균 등 특수한 유해물질 통과 저지

5. **호르몬 생산**:
   - 에스트로겐, 프로게스테론, 생식샘자극호르몬 분비

#### 태반에서 분비되는 호르몬

| 호르몬 | 종류 | 효과 |
|--------|------|------|
| **융모성 생식샘자극호르몬** (HCG) | 뇌하수체 유사 호르몬 | • LH와 유사한 작용<br/>• 임신 첫 5주 반: 모체의 황체 유지<br/>• 배아의 면역 거부 억제<br/>• TSH와 유사한 활동 |
| **융모성 몸젖샘자극호르몬** (HCS) | 뇌하수체 유사 호르몬 | • 프로락틴 + 성장호르몬과 유사<br/>• 모체: 지방분해 + 태아용 포도당 보존 촉진 |
| **프로게스테론** (progesterone) | 성호르몬 | • 임신 중 자궁속막 유지<br/>• 생식샘자극호르몬 분비 억제<br/>• 젖샘꽈리조직 발생 촉진 |
| **에스트로겐** (estrogen) | 성호르몬 | • 임신 중 자궁속막 유지<br/>• 생식샘자극호르몬 분비 억제<br/>• 젖샘 발생 촉진<br/>• 프로락틴 분비 억제<br/>• 옥시토신에 대한 자궁 감응성 촉진 |

#### 후산 (Afterbirth)
- **시점**: 출산 시 태아 출생 후
- **의의**: 태반이 자궁에서 떨어져 나옴

### 분만 (Labor and Delivery)

#### 분만의 정의
- **진통** (labor): 분만을 일으키는 모든 과정
- **분만** (delivery) 또는 **출생** (birth): 아기가 나오는 과정 자체

#### 분만 준비
- **분만 전 10~14일**: 태아의 머리가 골반 속에 위치
- **거짓진통** (가성진통, false labor): 분만 3~4주 전 발생 → 불규칙적인 자궁수축 특징
- **분만 임박의 신호**:
  - 자궁의 규칙적 수축
  - 자궁목에서의 점액 분비
  - 양막파열

#### 분만의 원인
- **원인**: 아직 명확하지 않음
- **시점**: 대개 태아가 모체에 의존 없이 살 수 있을 만큼 성장했을 때
- **기간**:
  - **초산** (첫 출산): 길어짐 (약 14시간)
  - **다산** (재출산): 점차 짧아짐 (7~8시간)

#### 분만의 단계 (3기)

**분만 1기 - 확장기** (Dilation Stage)
- **시간**: 6~12시간 (가장 길어짐)
- **과정**:
  1. 진성수축 나타남 → 자궁목 부분 확장 (지름 약 10 ㎝)
  2. 자궁 위쪽에서 약한 수축 시작 → 아래쪽으로 내려옴
  3. 수축: 조금씩 격렬해짐 + 빨라짐
  4. 매 수축마다 아이 머리 → 자궁목 부분 압박
  5. 자궁목: 부드러워짐 + 얇아짐 + 확장됨
  6. 양수 파열 → '파수(break the water)'

**분만 2기 - 태아 만출기** (Expulsion Stage)
- **시간**: 길게 2시간까지 (초산 약 50분, 재출산 약 20분)
- **과정**:
  1. 완전확장에서 태아 만출까지
  2. 아이: 자궁목 → 질 → 엄마 몸 밖으로 배출
  3. 자연분만 시: 배근육이 '용쓴다'는 동작으로 힘을 모아 씀

**분만 3기 - 태반기** (Placental Stage)
- **시간**: 보통 15분 이내에 완료
- **과정**:
  1. 태아 만출 직후 태반 만출
  2. 분만 후에도 자궁 수축 강하게 계속
  3. 혈관 눌림 → 혈액 공급 제한 → 태반이 자궁벽에서 분리
  4. 탯줄을 살짝 잡아당기면 후산 쉽게 제거
- **중요**: 태반 + 부속물 모두 제거 필요 (안 하면 자궁 출혈 지속)

**분만 4기 - 회복기** (Revival Stage)
- **시간**: 태반만출 후 1~4시간
- **특징**: 잠재적 위험 있는 시기
- **주의할 합병증**:
  - 자궁이완으로 인한 출혈
  - 요정체 (urinary retention)
  - 저혈압

## 추가 참고: 성적 표현과 개념

### 남성의 성반응주기 중 생식기의 변화
- **미흥분 상태**: 기본 상태
- **부분적 흥분 상태**: 색깔이 짙어짐
- **완전 발기**: 최대 팽대 (고환 확대 + 올라감)
- **극치기**: 정낭 수축 + 전립샘 확대 + 요도망울샘에서 정액 분비 + 고환이 크기 증가
- **사정**: 요도 수축 → 정액 사출 + 전립샘 수축 + 전립샘 조임근 수축 → 방광입구 폐쇄
- **해소기**: 음경으로부터의 혈액 방출 + 발기 소실, 고환으로부터 혈액 방출 + 축소, 근육이완 + 고환이 처짐

### 여성의 성반응주기 중 생식기의 변화
- **미흥분 상태**: 기본 상태
- **흥분기**: 바깥음부 색깔 변화, 자궁상승, 질확대
- **고조기**: 질의 하부 1/3 부종(음핵 포함), 자궁 상승 계속, 질 상부 2/3 확대
- **극치기**: 질 하부 1/3의 근육 수축, 자궁의 율동적 수축, 항문괄약근의 율동적 수축
- **해소기**: 생식기 반응의 점진적 회복 및 안정화
