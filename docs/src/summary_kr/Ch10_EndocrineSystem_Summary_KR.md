# Chapter 10. 내분비계통 (Endocrine System)

## 1. 뇌하수체

### 내분비계통의 개요
- **정의**: 내분비계통(endocrine system)은 순환계로 직접 방출되어 대사 및 신체과정을 조절하는 호르몬을 생산하는 곳
- **내분비샘**: 갑상선, 뇌하수체, 부신, 이자, 가슴샘, 부갑상샘, 고환(여: 난소) 등
- **호르몬(hormone)**: 그리스어의 '잠에서 깨어나게 하는 물질'에서 유래
- **호르몬의 생물학적 특징**:
  - 미량으로도 효력이 강한 유효물질
  - 동물의 종에 따르는 특이성이 없고 효과가 같음
  - 이종 동물에 주사하여도 항체(antibody)가 생기지 않는 경우가 많음
  - 단백질인 경우에도 비교적 구조가 간단하며 분자량도 작음
  - 성호르몬과 갑상샘호르몬 이외의 호르몬은 경구적으로 투여하면 소화효소에 의해 분해
  - 분비량이 많거나 적어도 생체에 유해하며, 하루 동안의 분비량은 체중, 연령, 성별, 육체적 활동에 따라 결정

### 뇌하수체 구조 (Pituitary body, Hypophysis)
- **위치**: 머리뼈안 바닥의 나비뼈 안장(터어키안, sella turcica) 속
- **크기**: 콩알 크기, 남자 0.5~0.6g, 여자 0.6~0.7g (여자가 약간 무거움)
- **연결**: 깔때기(누두, infundibulum)를 통해 셋째뇌실(제3뇌실) 바닥과 연접
- **구성**:
  - 샘뇌하수체(선뇌하수체, adenohypophysis) → 앞엽(anterior lobe) + 중간엽(pars intermedia)
  - 신경뇌하수체(neurohypophysis) → 뒤엽(posterior lobe) + 깔때기(infundibulum)

### 앞엽호르몬
- **갑상샘자극호르몬(TSH)**: 갑상샘 발육 촉진, 갑상샘호르몬(T₄) 분비 촉진, 음성되먹임 기전(negative feedback mechanism)
- **부신겉질자극호르몬(ACTH)**: 33개 아미노산 폴리펩티드, 부신겉질 내분비 활동 조절, 당류겉질호르몬 분비 촉진, 음성되먹임 기전
- **난포자극호르몬(FSH)**: 여성 — 난소 내 세포 활동·성숙 촉진, 난포호르몬(estrogen) 분비 촉진 / 남성 — 버팀세포(Sertoli's cell)에 작용하여 정자형성 촉진
- **황체형성호르몬(LH)**: 여성 — 배란 촉진, 황체 형성, 황체호르몬(progesterone) 분비 촉진 / 남성 — 사이질세포(Leydig's cell)에 작용하여 남성호르몬(testosterone) 분비 촉진 → 사이질세포자극호르몬(ICSH)이라고도 함
- **성장호르몬(GH)**: 뼈와 연부조직 성장 촉진, 당신생과 당원질 분해 촉진, 혈액 중 지방농도 상승. 몸자극호르몬(STH, somatotropin)이라고도 함. 과잉 분비 → 거인증(giantism), 성장 후 과잉 분비 → 말단비대증(acromegaly), 분비 결여 → 난쟁이증(dwarfism)
- **프로락틴(PRL)**: 유방만 표적기관, 분자량 약 25,000, 젖(유즙, milk) 분비 촉진. 황체자극호르몬(LTH)이라고도 함

### 뒤엽호르몬
- **항이뇨호르몬(ADH, vasopressin)**: 콩팥의 먼쪽곱슬세관과 집합세관에서 수분 재흡수 촉진, 이뇨(diuresis) 억제. 분비 부족 시 요붕증(diabetes insipidus) → 다량의 오줌 배설, 심한 갈증
- **옥시토신(OT)**: 자궁근육 수축 → 분만 촉진, 젖샘의 샘꽈리 벽 근육상피세포(myoepithelial cell) 수축 → 젖 배출

## 2. 갑상샘

### 갑상샘 구조 (Thyroid gland)
- **크기**: 약 30g, 사람의 내분비샘 중 가장 큼
- **위치**: 목 앞에서 기관의 위쪽
- **모양**: 나비가 날개를 펴고 있는 듯한 모양
- **구성**: 왼·오른쪽의 2엽(lobe) + 잘록부분(협부, isthmus)
- **조직학적 특징**: 소포(follicle) 구조, 요오드(iodine, I) 함유량이 큼
- **분비 호르몬**: 주로 티록신(thyroxine, T₄), 일부 삼요오드티로닌(triiodothyronine, T₃). 작용은 동일하나 효력은 T₃이 좀 더 강함. 부세포에서 티로칼시토닌(thyrocalcitonin)도 분비

### 갑상샘호르몬의 생성과 분비
- 음식물의 요오드 → 소화관 흡수 → 혈류 → 갑상샘 소포 샘세포가 능동적으로 세포 내로 끌어들임
- 생성된 갑상샘호르몬은 단백질과 결합한 형태로 콜로이드 내에 저장
- 분비 시 단백질과 분리되어 혈액 속으로 방출
- 뇌하수체의 갑상샘자극호르몬(TSH)에 의해 조절

### 갑상샘호르몬의 생리작용
- 물질대사의 촉진
- 발육·성장 촉진 (특히 뼈·치아·뼈대근육)
- 교감신경 활동 항진 → 심장박동수 증가, 혈관확장, 혈류량 증가, 땀 분비 촉진
- 지능 발달 촉진, 정신기능 자극
- 젖샘에서의 젖 분비 촉진
- 당질의 흡수 및 당신생 촉진 → 혈당 상승
- 쓸개즙에서의 콜레스테롤 배출 촉진

### 갑상샘기능저하 (Hypothyroidism)
- **어린아이**: 크레틴증(cretinism) — 난쟁이, 중추신경 발달 부진, 지각발달 지연, 백치
- **어른**: 점액수종(myxedema) — 정신적 둔감, 심장 박동 느림, 피부온도 저하, 피부 건조·탈모, 피부밑 점액성 물질 축적 → 부기(edema)

### 갑상샘기능항진 (Hyperthyroidism)
- 바세도병(Basedow's disease) = 그레이브병(Grave's disease)
- **3가지 증상**: 안구돌출, 빠른맥(빈맥), 갑상샘종
- 기초대사 항진, 식욕 증가, 체중 감소
- **치료**: 갑상샘 부분 절제 또는 방사성동위원소 요오드(I¹³¹) 사용

### 티로칼시토닌 (Thyrocalcitonin)
- 칼시토닌(calcitonin, CT)이라고도 함
- 혈액 중 칼슘 농도를 낮추는 작용 → 저칼슘호르몬(hypocalcemic hormone)
- 부갑상샘호르몬(고칼슘호르몬)과 대항작용
- 뼈파괴세포(파골세포, osteoclast) 활동 저하 → 뼈로부터 Ca²⁺ 유리 억제
- 뼈조직세포 활성도 증가 → Ca²⁺ 침착 → 혈액 Ca²⁺ 농도 저하

## 3. 부갑상샘

### 부갑상샘 구조 (Parathyroid gland)
- **위치**: 갑상샘 뒷면의 아래위에 각각 한 쌍씩, 모두 4개
- **크기**: 쌀알 정도
- **분비 호르몬**: 부갑상샘호르몬(parathormone, PTH)

### 부갑상샘호르몬의 기능
- 혈액 중 Ca²⁺ 농도를 높이는 고칼슘호르몬(hypercalcemic hormone)
- 뼈와 치아의 발육에 관여
- 부갑상샘호르몬 감소 → 혈중 칼슘 농도가 정상치의 1/2 정도로 감소 → 근육 경련 → 강직(tetany). 소아나 임산부에서 일어나기 쉬움
- 체액의 Ca²⁺이온 조절: 칼슘이온 감소 → 부갑상샘 자극 → PTH 분비 → 뼈의 칼슘이온을 체액으로 배출
- **혈액 Ca²⁺ 농도 조절**: 부갑상샘호르몬, 칼시토닌, 칼시트리올의 협동작용

## 4. 이자

### 이자 구조 (Pancreas)
- **위치**: 위의 뒤쪽, 첫째~둘째허리뼈 앞부분, 뒤쪽 배벽을 가로질러 수평
- **구성**: 머리, 몸통, 꼬리의 세 부분
- **기능**: 외분비(소화샘) + 내분비(호르몬)
- **이자섬(랑게르한스섬, Langerhans's islet)**: 약 100만 개, 이자꼬리에 많음, 이자 전체 무게의 1~2%
  - α세포(10~30%): 아닐린에 붉은색 염색, 글루카곤(glucagon) 분비
  - β세포(60~90%): 아닐린에 청자색 염색, 인슐린(insulin) 분비
  - δ세포(1~8%): 과립 없음, 소마토스타틴(somatostatin) 분비

### 인슐린 (Insulin)
- β세포에서 분비되는 단백질성 호르몬
- **생리작용**:
  - 혈당 저하: 조직 세포에 포도당이 들어가기 쉽게 함
  - 과다분극(hyperpolarization): 세포막의 안정막전위 증가, 세포내 K⁺ 농도 높이고 세포외액 K⁺ 농도 낮춤
  - 당의 지방전환 촉진
  - 단백질 합성 촉진

### 당뇨병 (Diabetes mellitus)
- 인슐린 부족에 기인하는 고혈당(hyperglycemia)
- **I형 당뇨병(IDDM)**: 인슐린의존당뇨병, 이자 β세포 파괴, 주로 소아 발생, 인슐린만 치료 가능
- **II형 당뇨병(NIDDM)**: 인슐린비의존당뇨병, 각 세포들의 인슐린 반응성 저하, 조직 민감도 감소
- **증상**: 다뇨증(polyuria), 다음증(polydipsia), 다식증(polyphagia), 체중감소, 성욕감퇴

### 글루카곤 (Glucagon)
- α세포에서 분비, 29개 아미노산 폴리펩티드
- 혈당 상승 작용, 인슐린에 길항 작용
- 주요 표적기관: 간(liver)
- 너무 적거나 많이 분비되어도 장애를 일으키지 않음

## 5. 부신

### 부신 구조 (Adrenal gland)
- **위치**: 왼·오른쪽 한 쌍, 콩팥 위에 부착
- **겉질(cortex)**: 황색, 비교적 단단한 조직
- **속질(medulla)**: 암갈색, 비교적 연함, 크롬친화성세포(chromaffin cell)

### 부신속질호르몬
- **에피네프린(epinephrine, adrenaline)**: 사람이나 개에서 주로 분비
- **노르에피네프린(norepinephrine, noradrenaline)**: 에피네프린의 전구물질, 교감신경 절이후섬유 말단 전달물질. 고양이에서 주로 분비
- **카테콜아민(catecholamine)**: 에피네프린 + 노르에피네프린 + 도파민(dopamine). 페닐알라닌(phenylalanine)으로부터 생합성. 노르에피네프린이 메틸화 → 에피네프린
- **에피네프린의 주요 작용**: 심장박동수 증가, 수축력 증가(강력), 뼈대근육 혈관 확장, 기도 확장, 당원질 분해 촉진 → 혈당 증가
- **노르에피네프린의 주요 작용**: 심장박동수 증가(약함), 혈관수축으로 혈압 크게 증가, 뼈대근육·피부·내장의 미세동맥 수축

### 부신겉질호르몬
- 부신겉질 기능항진 → 쿠싱증후군(Cushing syndrome), 기능저하 → 애디슨병(Addison's disease)
- **토리층(사구대, zona glomerulosa)**: 가장 바깥층, 염류겉질호르몬(mineralocorticoids) 분비 → 알도스테론(aldosterone) — Na⁺ 재흡수 촉진, 체액 조절
- **다발층(속상대, zona fasciculata)**: 중간층, 가장 두꺼움(부신겉질의 ⅔ 이상), 당류겉질호르몬(glucocorticoids) 분비 → 코티솔(cortisol), 코티손(cortisone)
- **그물층(망상대, zona reticularis)**: 가장 속층, 안드로겐(androgen) + 소량의 당류겉질호르몬 분비

### 당류겉질호르몬의 주요 작용
- 지방과 단백질을 당질로 전환 → 당신생(gluconeogenesis)
- 항염증 작용 → 류마티스성 질환 치료
- 림프구, 호산구, 호염구 감소 / 호중구, 적혈구, 혈소판 증가
- 위액의 산 및 펩신 분비 촉진, 점액 분비 억제 → 위점막 저항성 감소
- 혈액 내 포도당 수준 증가 → 정상 세포대사 촉진
- 프로스타글란딘 억제 → 통증 감소
- 스트레스에 대한 반응: 내부 환경 항상성 유지

## 6. 소화관호르몬

### 소화관호르몬 개요
- 소화관의 활동(소화액 분비, 소화관 운동)을 촉진하거나 억제하는 호르몬
- 소화관 점막의 특수한 세포에서 분비 → 위장호르몬(gastrointestinal hormone, GI hormone)

### 세크레틴 (Secretin)
- 샘창자(십이지장) 점막에서 분비
- 이자에 작용하여 HCO₃⁻이온을 많이 함유한 이자액 분비 촉진 (소화효소 함량은 적음)

### 가스트린 (Gastrin)
- 위의 날문부점막의 G세포(gastric containing cell)에서 분비
- 위액 분비 촉진, 위 운동 촉진

### 콜레시스토키닌-판크레오지민 (CCK-PZ)
- 음식물이 샘창자로 내려오면 분비
- 콜레시스토키닌(CCK): 쓸개(담낭) 수축
- 판크레오지민(PZ): 소화효소가 많고 HCO⁻이온이 적은 이자액 분비 촉진

### 기타 소화관호르몬
- **위억제폴리펩티드(GIP)**: 위의 염산 분비 억제, 이자의 인슐린 분비 촉진
- **모틸린(motilin)**: 위와 작은창자의 운동 촉진
- **혈관작용창자펩티드(VIP)**: 소화관 운동 억제, 혈압 강하

## 7. 호르몬 관련 물질

### 심방 나트륨 배설촉진 펩티드 (ANP)
- 안지오텐신-II와 반대로 콩팥에서 수분과 나트륨이온 제거
- 항이뇨호르몬이나 알도스테론 분비 억제 → 혈액량과 혈압 감소

### 칼시트리올 (Calcitriol)
- 부갑상샘호르몬의 자극에 의해 분비되는 스테로이드호르몬
- 창자관에서 칼슘과 인의 흡수 촉진

### 적혈구생성소 (Erythropoietin, EPO)
- 콩팥의 구성 세포에 산소 부족 시 분비되는 펩티드호르몬
- 적색골수를 자극하여 적혈구생성 촉진

### 레닌 (Renin)
- 혈압 저하 시 분비
- 안지오텐신계를 자극하여 혈압 상승 유도

### 멜라토닌 (Melatonin)
- 솔방울샘(송과체, pineal body)에서 분비
- 빛 감각과 관계, 생식기 발육 억제
- 혈중 농도: 낮보다 밤에 10배 증가

### 프로스타글란딘 (Prostaglandin, PG)
- 여러 형(A, B, C, D, E, F, G) 존재
- 혈관 확장, 민무늬근 수축 또는 이완 (특히 자궁과 기관지근육)
- 혈소판 응집 촉진 또는 저지, 황체 퇴행, 자궁 수축

### 인히빈 (Inhibin)
- 고환의 버팀세포와 난소의 과립세포에서 분비되는 당단백질
- 인히빈A와 인히빈B
- 뇌하수체에 작용하여 난포자극호르몬(FSH) 분비 억제

### 렙틴 (Leptin)
- 지방조직에서 분비되는 단백질호르몬, 시상하부에서 작용
- 포만감 제공, 식사 후 20분이 지나야 분비 시작
- 비만한 사람의 혈액에 렙틴이 많음
