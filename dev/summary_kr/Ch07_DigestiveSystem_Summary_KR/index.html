Chapter 07. 소화계통 (Digestive System)

1. 소화관 벽의 구조

소화계통 개요

  • 정의: 섭취된 음식물을 더 작은 분자로 분해한 후 혈액으로 들어오게 하는 것을 주된 기능으로 하는 계통
  • 소화(digestion): 음식물의 복잡하고 큰 성분분자를 단순하고 작은 분자로 분해하는 과정
  • 흡수(absorption): 분해된 산물을 혈액 내로 이동시키는 과정
  • 소화관(digestive tube): 입에서 항문에 이르는 긴 관(약 9m) → 입안, 인두, 식도, 위, 작은창자, 큰창자
  • 소화샘(digestive gland): 침샘, 간, 이자

소화관 벽의 4층 구조

  • 점막층(mucosa): 표면 상피세포 + 고유판(lamina propria) + 점막근육층(muscularis mucosa) → 점액과 소화효소 분비, 방어 및 흡수
  • 점막밑층(submucosa): 분비샘, 혈관, 림프관, 신경, 성긴결합조직 → 영양 공급, 흡수된 물질 운반
  • 근육층(muscular layer): 돌림근육(윤주근) + 세로근육(종주근) → 소화관과 내용물의 이동(꿈틀·분할운동)
  • 장막층(serosa): 상피세포 + 결합조직 → 장액 분비, 방어, 윤활

소화관의 신경지배

  • 부교감신경: 소화계통의 활성을 증가시킴 → 미주신경 가지를 통해 식도, 위, 이자, 쓸개, 작은창자, 큰창자 몸쪽 부분으로 전달
  • 교감신경: 부교감신경 작용과 반대 → 혼합운동과 꿈틀운동 억제, 조임근 수축으로 물질 이동 억제

2. 소화기의 구조 — 입안·인두·식도

입안(oral cavity)

  • 기능: 음식물 받아들임, 씹기, 미각, 발성
  • 구성: 위·아래입술, 볼, 입술틈새(구열), 입꼬리(구각)
  • 입천장(palate): 앞쪽 2/3 = 단단입천장(경구개, hard palate), 뒤쪽 1/3 = 물렁입천장(연구개, soft palate)
  • 목젖(uvula): 물렁입천장 뒤쪽 중앙에 위치

혀(tongue)

  • 분계고랑(terminal sulcus)을 경계로 혀몸통(body) + 혀뿌리(root) + 혀끝(apex)
  • 혀의 유두:
    • 실유두(filiform papillae): 수가 가장 많음, 맛봉오리 없음
    • 버섯유두(fungiform papillae): 불규칙 산재, 약간의 맛봉오리
    • 성곽유두(circumvallate papillae): 분계고랑 앞에 7~12개, 많은 맛봉오리
    • 잎새유두(foliate papillae): 혀뿌리 가쪽면, 소아·사춘기까지만 존재 후 퇴화

치아(teeth)

  • 구조: 치아머리(치관) + 치아목(치경) + 치아뿌리(치근)
  • 주성분: 상아질(dentin) → 사기질(enamel, 가장 단단한 조직)이 치아머리를 덮음, 시멘트질(cementum)이 치아뿌리를 덮음
  • 경도 순서: 사기질 > 상아질 > 시멘트질

침샘(salivary gland)

  • 귀밑샘(parotid gland): 가장 큰 샘, 순장액성 침 분비, 볼거리(mumps) 발생
  • 턱밑샘(submandibular gland): 혼합성 침 분비
  • 혀밑샘(sublingual gland): 점액이 많은 걸쭉한 침 분비

인두(pharynx)

  • 입안과 식도 사이, 길이 약 12cm
  • 인두편도(pharyngeal tonsil) 비대 시 아데노이드(adenoid)

식도(esophagus)

  • 길이 약 25cm, 세 군데 잘록부분(협착부)
    • 첫째: 식도 시작 부위(앞니에서 13~15cm)
    • 둘째: 기관 갈림 부위(앞니에서 22~25cm)
    • 셋째: 가로막 관통 부위(앞니에서 38~40cm)
  • 근육 구성: 위쪽 1/3 뼈대근육, 중앙 1/3 혼합, 아래쪽 1/3 민무늬근육

3. 위(Stomach)

위의 구조

  • 배안 왼쪽 위배부분, 용적 1.0~2.5L
  • 부위: 들문(분문, cardia) → 위바닥(위저, fundus) → 위몸통(위체, body) → 날문(유문, pylorus)
  • 날문조임근(pyloric sphincter): 음식물의 샘창자 배출 조절
  • 큰굽이(greater curvature): 왼쪽 아래의 커다란 가쪽면
  • 작은굽이(lesser curvature): 오른쪽 위의 짧은 가쪽면
  • 모패임(angular notch): 암과 궤양 발생이 쉬운 곳
  • 근육층 3층: 바깥세로근(외종주근) + 중간돌림근(중윤주근) + 속빗근(내사주근)

위샘(gastric gland)

  • 점액목세포(mucous neck cell): 점액(mucin) 분비 → 위벽 보호
  • 으뜸세포(chief cell): 펩시노겐(pepsinogen) 분비, 레닌·리파아제도 분비
  • 벽세포(parietal cell): 가장 큰 샘세포, 염산(HCl)과 위내인자(endogastric factor) 분비 → 위내인자 부족 시 악성빈혈
  • 친크롬은세포(argentaffin cell): 세로토닌(serotonin) 분비 추정

4. 작은창자(Small Intestine)

개요

  • 지름 2~3cm, 길이 6~7m → 샘창자(십이지장) + 빈창자(공장) + 돌창자(회장)
  • 기능: 쓸개즙, 이자액, 창자액을 혼합하여 소화 완료, 영양분 흡수

샘창자(duodenum)

  • 길이 25cm, C자형, 창자간막 없음
  • 4부위: 위부분, 내림부분, 수평부분, 오름부분
  • 내림부분에 큰샘창자유두 → 온쓸개관 + 이자관 개구

빈창자(jejunum)와 돌창자(ileum)

  • 창자간막에 의해 뒤배벽에 매달림
  • 빈창자 = 샘창자 연결 2/5(주로 왼쪽 위배), 돌창자 = 나머지 3/5(오른쪽 아래배)

작은창자벽의 구조

  • 점막: 돌림주름(circular fold) + 창자융모(intestinal villi) → 모세혈관 + 중심암죽관으로 영양분 흡수
  • 창자샘(Lieberkuhn gland): 융모 사이에 위치, 바닥에 파네트세포(Paneth's cell)
  • 점막밑층: 샘창자샘(Brunner's gland, 알칼리성 점액), 돌창자에 파이어판(Peyer's patch)
  • 근육층: 속돌림층 + 바깥세로층 → 분할운동·꿈틀운동

5. 큰창자(Large Intestine)

개요

  • 길이 약 1.5m, 주된 작용: 수분과 염분 흡수

막창자(cecum)

  • 소화관 중 가장 굵음, 길이 7~8cm
  • 막창자꼬리(충수, appendix): 길이 6~7cm
  • 돌막창자판막(ileocecal valve): 역류 방지
  • 맥버니 점(McBurney's point): 배꼽~오른위앞엉덩뼈가시 연결선의 가쪽 1/3

잘록창자(colon) 4부위

  • 오름창자(ascending colon): 길이 20cm, 잘록창자 중 가장 짧음
  • 가로창자(transverse colon): 길이 약 50cm, 잘록창자 중 가장 김
  • 내림창자(descending colon): 길이 약 25cm
  • 구불창자(sigmoid colon): 길이 약 45cm, S자 형태

곧창자(rectum)

  • 길이 약 10cm
  • 곧창자정맥얼기에 울혈 → 치핵(hemorrhoid)

큰창자의 구조적 특징

  • 잘록창자띠(taenia coli): 세로근층 3개 다발
  • 잘록창자팽대(haustra): 볼록한 주머니
  • 큰창자점막에는 창자융모 및 돌림주름이 없음
  • 술잔세포(goblet cell): 점액 분비로 배변 원활

6. 간·쓸개·이자

간(liver)

  • 인체에서 가장 큰 샘, 무게 약 1.5kg
  • 간엽: 왼엽 + 오른엽(더 두껍고 큼) + 네모엽 + 꼬리엽
  • 간소엽(hepatic lobule): 간의 기본 구조 단위, 중심정맥에서 방사되는 간세포줄
  • 별세포(Kupffer cell): 굴모세혈관 내피에 산재, 포식작용
  • 쓸개즙 경로: 간세포 → 온간관 → (+ 쓸개관) → 온쓸개관 → 큰샘창자유두
  • 오디의 조임근(sphincter of Oddi): 큰샘창자유두 개구부의 조임근

쓸개(gallbladder)

  • 간의 오른엽 아래, 용적 약 70mL
  • 쓸개즙 일시 저장
  • 쓸개돌(gallstone) 자주 발생

이자(pancreas)

  • 위 뒤쪽, 무게 약 70g → 이자머리(췌두) + 이자몸통(췌체) + 이자꼬리(췌미)
  • 외분비샘: 소화효소 하루 약 1,200mL → 트립신(단백질), 아밀라아제(탄수화물), 리파아제(지방)
  • 내분비샘: 이자섬(랑게르한스섬) 약 100만 개
    • α세포 → 글루카곤(glucagon)
    • β세포(75%) → 인슐린(insulin)
    • δ세포 → 소마토스타틴(somatostatin)

복막(peritoneum)

  • 내장쪽복막(visceral peritoneum) + 벽쪽복막(parietal peritoneum) → 복막안(peritoneal cavity)
  • 큰그물막(greater omentum): 위의 큰굽이에서 드리워짐
  • 작은그물막(lesser omentum): 간과 위의 작은굽이 사이
  • 복막뒤장기(retroperitoneal organ): 샘창자, 이자, 콩팥, 오름창자, 내림창자 등

7. 소화 생리 — 입안·위

입안 내의 소화

  • 씹기(mastication): 주로 삼차신경 지배, 입술·뺨은 얼굴신경, 혀는 혀신경
  • 침(saliva): 성인 하루 1.0~1.5L 분비
    • 프티알린(ptyalin): 전분 → 덱스트린 + 맥아당
    • 점액(mucin): 음식물 삼킴 보조
    • 파로틴(parotin): 귀밑샘에서 분비, 뼈와 치아 석회침착 촉진
  • 침의 생리작용: 소화작용, 삼킴작용, 습윤·윤활작용, 청정작용, 배설작용, 체온조절작용, 수분대사 조절

삼킴(swallowing)

  • 삼킴중추: 숨뇌
  • 제1기(입안기): 수의운동
  • 제2기(인두기): 불수의 반사운동
  • 제3기(식도기): 불수의 꿈틀운동 → 식도입구~위 들문까지 약 5초

위에서의 소화

  • 위의 운동: 꿈틀운동 주기 약 20초, 부교감신경 촉진·교감신경 억제
  • 암죽(chyme): 위 내용물이 위액과 혼합된 반유동 상태
  • 위 내용물 배출: 보통 식후 2~3시간(정상 3~4시간)
  • 작은창자—위반사(enterogastric reflex): 샘창자에 암죽 많아지면 위 운동·분비 억제

위액(gastric juice)

  • 하루 1.0~1.5L, pH 0.92~1.58(강산성)
  • 펩신(pepsin): 단백질 → 프로테오스, 펩톤
  • 리파아제(lipase): 중성지방 → 지방산 + 글리세린
  • 레닌(rennin): 카세이노겐 → 카세인(젖 응고), 유아기에 많음
  • 염산(HCl): 펩시노겐 활성화, 최적 pH 유지, 살균, 쓸개즙·이자액 분비 촉진

위액 분비의 3단계

  • 뇌기(cephalic phase): 전체 위액의 30~40%
  • 위기(gastric phase): 위액의 40~50%
  • 창자기(intestinal phase): 위액의 약 5%

8. 소화 생리 — 장·간 기능

작은창자의 소화

  • 분절운동(segmentation): 수축·이완 교대 → 음식물과 소화액 혼합
  • 꿈틀운동(peristalsis): 파도형 수축운동 → 음식물을 하부로 이동, 큰창자 도달까지 3~5시간
  • 흔들운동(pendular movement): 세로근육 수축·이완 → 혼합·수송

이자액(pancreatic juice)

  • 하루 약 500~800mL, pH 8.5(약알칼리성)
  • HCO₃ 이온 → 위산 중화
  • 트립신·카이모트립신: 단백질 → 폴리펩티드 (엔테로키나아제에 의해 활성화)
  • 리파아제: 중성지방 → 지방산 + 글리세린
  • 아밀라아제: 전분 → 이당류 / 말타아제: 맥아당 → 포도당 / 락타아제: 유당 → 포도당 + 갈락토오스
  • 분비 조절: 세크레틴(secretin) → NaHCO₃ 분비, 콜레시스토키닌-판크레오지민 → 소화효소 + 쓸개즙 분비 촉진

쓸개즙(bile)

  • 1일 500~1,000mL 배출
  • 쓸개즙산: 지방 유화(emulsification) → 리파아제 작용 촉진, 지방산 흡수 촉진
  • 쓸개즙색소: 빌리루빈(bilirubin) → 적혈구 파괴 시 혈색소 분해산물
  • 기타: 지용성 비타민(A, D, E, K) 흡수 촉진, 철·칼슘 흡수 촉진

큰창자의 소화

  • 큰창자액: 알칼리성 점액, 소화효소 거의 없음
  • 식후 4시간 → 큰창자 진입, 12시간 → 구불창자 도달
  • 집단꿈틀운동(mass peristalsis): 위—잘록창자반사에 의해 1일 2~3회

배변

  • 속항문조임근(internal anal sphincter): 반사적 이완(불수의근)
  • 바깥항문조임근(external anal sphincter): 수의근육 → 의지로 수축 가능
  • 배변중추: 척수, 상위중추: 대뇌겉질

간의 기능

  • 탄수화물 대사(혈당 유지), 지방대사(인지질·콜레스테롤 합성), 단백질 대사(알부민·섬유소원 합성)
  • 암모니아 → 요소 합성(오르니틴 회로) → 콩팥 배설
  • 약물·유해물질 해독, 호르몬 불활성화
  • 열 생산(몸의 주된 열 생산 기관)
  • 물질 저장: 당원질, 지용성 비타민(A, D, E, K), 철분, 구리, 비타민 B₁₂
  • 별세포(Kupffer cell): 세균·이물질 포식, 면역항체 생산
