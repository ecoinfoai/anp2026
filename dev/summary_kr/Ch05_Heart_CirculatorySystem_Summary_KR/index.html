Chapter 05. 심장·순환계통 (Heart & Circulatory System)

1. 심장

심장의 개요 (Heart)

  • 정의: 심장(heart)은 근육으로 된 이중 펌프로서 정맥혈액을 허파로 보내고, 동맥혈액을 신체 각 부분으로 보내는 순환작용을 일으키는 속빈기관(hollow organ)
  • 위치: 가슴안 속, 두 겹의 심장막(pericardium)에 싸여 있으며, 2/3가 정중선보다 왼쪽에 치우쳐 있음
  • 크기: 무게 약 250~300g, 길이 약 14㎝, 폭 10㎝, 두께 8㎝, 자신의 주먹보다 약간 큰 크기
  • 구조명칭:
    • 심장축(심축, axis of the heart): 심장의 긴축
    • 심장바닥(심저, base of the heart): 심장 위끝의 넓은 부위
    • 심장꼭대기(심첨, apex of the heart): 심장 아래끝 왼위쪽의 뾰족한 부위
    • 심장고랑(관상고랑, coronary groove): 위쪽 1/3 부위에 가로로 달리는 잘록한 고랑

심장의 내부구조

  • 오른심방(우심방, right atrium): 위대정맥(상대정맥)과 아래대정맥(하대정맥)을 통해 정맥혈액을 받음. 위대정맥은 정맥판막이 없음. 타원오목(난원와, fossa ovalis)이 심방사이막에 있으며, 태아의 타원구멍(난원공)이 출생 후 폐쇄된 흔적임
  • 오른심실(우심실, right ventricle): 수축으로 혈액을 허파동맥을 통해 허파로 운반. 힘줄끈(건삭)이 꼭지근육(유두근)에 연결. 허파동맥 바로 아래의 원뿔(원추, cone) 구조가 존재. 반달판막(반월판, semilunar valves)이 혈액 역류 방지
  • 왼심방(좌심방, left atrium): 4개의 허파정맥(pulmonary veins)을 통해 동맥혈을 받음. 벽이 얇고 왼심방귀(좌심이)가 붙어 있음
  • 왼심실(좌심실, left ventricle): 벽 두께 약 2㎝, 오른심실벽 두께의 3배. 대동맥을 통해 온몸으로 산화된 혈액을 내보냄. 이첨판막(bicuspid valve) = 승모판막(mitral valve)으로 혈류 조절

심장의 판막

  • 이첨판막(bicuspid valve): 왼방실구멍, 2개의 첨판(cusp), 승모판막(mitral valve)이라고도 함
  • 삼첨판막(tricuspid valve): 오른방실구멍, 3개의 첨판
  • 허파동맥판막(pulmonary valve): 허파동맥 입구, 심실 이완 시 혈액 역류 방지
  • 대동맥판막(aortic valve): 대동맥 입구, 허파동맥판막보다 훨씬 큰 크기
  • 방실판은 힘줄끈(건삭, chordae tendineae)에 의해 꼭지근육(papillary muscle)에 연결

심장근육의 기능적 특징

  • 결절조직, 심장전도근육섬유(Purkinje's fiber)조직, 가로무늬근육조직으로 구성
  • 4가지 특성: 율동성(rhythmicity), 전도성(conductivity), 수축성(contractility), 흥분성(excitability)
  • 심장벽의 3층: 심장속막(심내막, endocardium) — 심장근육층(심근층, myocardium) — 심장바깥막(심외막, pericardium)
  • 심장근육조직은 기능적 세포결체(functional syncytium)를 형성

심장의 자극전도계

  • 굴심방결절(동방결절, SA node) → 방실결절(AV node) → 방실다발(방실속, AV bundle) → 방실다발가지(좌·우각) → 심장전도근육섬유(푸르킨예섬유, Purkinje fiber) → 심실근육 수축
  • 심장박동(cardiac pulse): 심장의 율동적인 수축
  • 심장주기(cardiac cycle): 안정 상태에서 약 0.8초
  • 심전도(EKG): 심장근육에서 일어나는 전기적 변동을 그래프로 나타낸 것

2. 혈관

혈관의 구조

  • 혈관은 속막(내막, tunica intima), 중간막(중막, tunica media), 바깥막(외막, tunica adventitia)의 3층 구조
  • 속막: 단층 혈관 내피세포로 덮여 있음
  • 중간막: 가장 두꺼우며, 고리 모양의 민무늬근육섬유와 탄력섬유
  • 바깥막: 결합조직이 주성분, 소량의 근육섬유 혼합

혈관의 종류

  • 동맥: 정맥보다 두꺼우며 높은 관류압 유지, 중간막이 두꺼워 혈관 모양 유지
  • 정맥: 동맥보다 혈관벽이 얇고 탄력성이 적음. 정맥판막(venous valves)이 역류 방지. 혈액 저장 역할
  • 모세혈관: 세동맥과 세정맥을 연결, 굵기 7~10㎛. 내피세포만의 속막만 존재. 단백질을 제외한 모든 액체 성분이 통과 가능

허파순환

  • 소순환(lesser circulation): 오른심실 → 허파동맥 → 허파 → 허파정맥 → 왼심방
  • 허파동맥에는 CO₂ 함량이 많고 O₂ 함량이 적은 정맥혈액
  • 허파정맥에는 동맥혈액

3. 온몸순환의 동맥

대동맥

  • 왼심실에서 출발 → 오름대동맥(ascending aorta) → 대동맥활(aortic arch) → 내림대동맥(descending aorta)
  • 내림대동맥: 가로막을 경계로 가슴대동맥(흉대동맥)과 배대동맥(복대동맥)

대동맥활의 가지

  • 팔머리동맥(brachiocephalic trunk): 오른온목동맥과 오른빗장밑동맥으로 분지
  • 왼온목동맥, 왼빗장밑동맥 순서로 3개의 분지

목동맥팽대, 목동맥토리, 대동맥토리

  • 목동맥팽대(carotid sinus): 속목동맥 시작부의 약간 부푼 부위, 압력수용체(baroreceptor), 혈압 상승 시 심장박동수 감소
  • 목동맥토리(carotid body): 온목동맥 분기점의 화학수용체, 혈액의 O₂ 감시, 호흡조절
  • 대동맥토리(aortic body): 대동맥 옆의 화학수용체, CO₂ 분압 증가·pH 저하에 반응

대뇌동맥고리

  • 대뇌동맥고리(circle of Willis): 속목동맥과 척추동맥이 뇌바닥에서 형성
  • 앞대뇌동맥, 중간대뇌동맥, 뒤대뇌동맥, 앞교통동맥, 뒤교통동맥으로 구성

4. 정맥계통

심장정맥

  • 심장정맥(coronary vein) → 심장정맥굴(coronary sinus) → 오른심방

위대정맥

  • 머리·목·팔, 가슴우리에서 혈액을 받음
  • 왼·오른팔머리정맥이 합쳐져 형성
  • 직경 약 2㎝, 아래대정맥 다음으로 큰 정맥

아래대정맥

  • 인체에서 가장 큰 정맥
  • 벽쪽가지: 허리정맥(4쌍), 아래가로막정맥
  • 내장쪽가지: 콩팥정맥, 부신정맥, 고환/난소정맥, 간정맥

문정맥

  • 위, 창자, 이자, 지라, 쓸개의 정맥혈이 모여 형성
  • 간 속의 굴모세혈관(sinusoid)을 거쳐 간정맥 → 아래대정맥
  • 혈액이 모세관을 두 번 지나는 유일한 경우

5. 태아순환

출생 전의 순환

  • 배꼽정맥(umbilical vein): 태반에서 산소·영양물질 풍부한 혈액을 태아에게 공급
  • 배꼽동맥(umbilical arteries): 태아의 정맥혈액을 태반으로 보내는 1쌍의 혈관
  • 타원구멍(foramen ovale): 심방사이막의 구멍, 오른심방 → 왼심방
  • 동맥관(ductus arteriosus): 허파동맥과 대동맥활 사이를 연결
  • 정맥관(ductus venosus): 배꼽정맥과 아래대정맥을 연결

출생 후의 흔적

  • 타원구멍 → 타원오목(난원와)
  • 동맥관 → 동맥관인대(ligamentum arteriosum)
  • 배꼽동맥 → 배꼽동맥인대
  • 배꼽정맥 → 간원인대(round ligament)
  • 정맥관 → 정맥관인대(ligamentum venosus)

6. 심장의 기능

심전도 파형

  • P파: 심방의 탈분극
  • QRS복합파: 심실의 탈분극, 약 0.08초 지속
  • T파: 심실의 재분극, 약 0.27초 지속
  • P-Q 간격: 굴심방결절에서 방실결절로 흥분 이동 소요 시간
  • S-T 분절: 심실근육에서 재분극이 지속되는 상태

심장음

  • 제1심장음: 방실판막(이첨판막·삼첨판막)이 닫힐 때, 낮고 둔한 소리
  • 제2심장음: 대동맥판막과 허파동맥판막이 닫히는 소리, 높고 예리한 소리
  • 심장잡음(heart murmurs): 판막 기능 이상 시 나타남

심장주기

  • 수축기(systole) + 이완기(diastole) = 심장주기(약 0.8초)
  • 심방수축기 0.11초, 심실수축기 0.27초, 심실이완기 0.42초
  • 스타링의 심장법칙(Starling's law): 심실 충만도에 따라 수축력 증가, 신경계통 관여 없음

심장의 수축력과 박출량

  • 심장박동수: 안정 시 약 70~80회/분
  • 1회 박출량: 안정 시 약 70㎖
  • 분당 박출량: 약 5ℓ (70㎖ × 70회/분)

심장의 신경과 반사

  • 교감신경: 심장박동수 증가, 수축 증강, 전도속도 촉진
  • 부교감신경(미주신경): 심장박동수 감소, 수축력 저하, 전도속도 지연
  • 베인브리지 반사: 대정맥 내 압력 증가 → 박동수 증가
  • 대동맥활·목동맥팽대 신경반사: 혈압 상승 → 압력수용체 → 박동수 감소
  • 목동맥토리 반사: CO₂ 농도 증가 감지 → 박동수 증가
  • 안구압박반사(Aschner reflex): 안구 압박 → 심장 억제중추 흥분 → 심장활동 억제

7. 혈압

혈압

  • 혈압(blood pressure, BP): 혈관 속의 압력, 심장에 가까울수록 높고 대정맥에서 거의 0
  • 수축기혈압(최고혈압), 이완기혈압(최저혈압)
  • 맥압(pulse pressure): 최고혈압 - 최저혈압, 보통 최고혈압의 1/3
  • 20~30세 정상혈압: 최고 약 120㎜Hg, 최저 약 80㎜Hg

혈압 측정

  • 보통 위팔동맥(상완동맥)에서 측정
  • 수은혈압계: 커프스 → 청진기로 코로트코프음(Korotkoff sound) 확인
  • 맥박(arterial pulse): 보통 노동맥(요골동맥)에서 촉지

혈관운동의 조절

  • 신경성 조절: 교감신경(혈관수축, 혈압 상승) + 부교감신경(혈관확장, 혈압 하강) = 혈관운동신경
  • 화학물질에 의한 조절: 탄산가스·젖산 → 혈관 확장, 아드레날린 → 혈관 수축, 히스타민 → 혈관 확장
  • 체액성 조절: 레닌-안지오텐신 계통 → 안지오텐신 II → 혈관수축 + 알도스테론 분비 → Na⁺ 재흡수 증가 → 혈압 상승

8. 림프계통

림프관

  • 모세림프관 → 림프관 → 오른림프관(우림프관) + 가슴림프관(흉관) → 빗장밑정맥과 아래대정맥의 정맥각으로 유입
  • 가슴림프관: 가로막 위오른쪽을 제외한 전신, 길이 약 40㎝, 지름 3~4㎜

림프절

  • 직경 2~30㎜, 난원형, 들림프관(수입림프관)으로 들어와 날림프관(수출림프관)으로 나감
  • 기능: ① 여과 및 포식작용 ② 림프구 생산 ③ 항체 형성

지라(비장, spleen)

  • 위치: 배안의 위왼쪽, 왼쪽 여덟째~열한째갈비뼈 뒷부분, 무게 약 170g
  • 백색속질(white pulp): B세포, T세포 분포, 항체생산 및 면역반응
  • 적색속질(red pulp): 혈액 저장, 순환혈액량 조절, 노화 적혈구 포식

가슴샘(흉선, thymus)

  • 가슴세로칸(종격) 속, 대동맥활 앞쪽, 복장뼈 윗부분 뒤쪽에 위치
  • 어릴 때 상대적으로 더 큼(25~30g), 사춘기 이후 크기 감소(10~15g)
  • 타이모신(thymosins) 분비 → T세포 성숙
  • 하살소체(Hassall's body): 속질의 각질화된 상피세포

편도(tonsil)

  • 림프소절의 집합체, 소화기·호흡기 입구를 세균으로부터 보호
  • 목구멍편도(palatine tonsil): 입안 깊숙이 위치
  • 인두편도(pharyngeal tonsil): 인두의 천장
  • 혀편도(lingual tonsil): 혀의 점막
  • 왈데이어 편도고리(Waldeyer's tonsil ring): 세 편도의 둥근 배열
