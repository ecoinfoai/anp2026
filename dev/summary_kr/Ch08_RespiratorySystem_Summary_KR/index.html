Chapter 08. 호흡계통 (Respiratory System)

1. 호흡기의 구조와 기능

호흡의 정의

  • 호흡(respiration): 외부로부터 산소를 몸 안으로 흡입하여 세포로 운반하고, 세포 대사 결과 발생한 이산화탄소를 몸 밖으로 배출하는 전 과정
  • 외호흡(external respiration): 허파 혈관과 허파꽈리 사이에서 산소를 받아들이고 이산화탄소를 내보내는 공기의 교환
  • 내호흡(internal respiration): 모세혈관 내 혈액과 조직세포 사이에서 공기가 교환되는 과정

위기도 — 코 (Nose)

  • 코(외비, nose)는 얼굴 중앙에 돌출한 3각원뿔 모양의 부위
  • 콧구멍(nostril, 외비공)은 공기가 코안으로 드나드는 출구
  • 코털(비모)은 공기 중 큰 입자를 막고 먼지를 거르는 작용

위기도 — 코안 (Nasal Cavity)

  • 코안(비강, nasal cavity)은 코 뒤쪽의 빈 공간으로, 코중격(비중격, nasal septum)에 의해 좌우로 나뉨
  • 코안은 벌집뼈의 체판(사판, cribriform plate)에 의해 머리뼈안과 분리되고, 입안과는 단단입천장으로 분리
  • 코안의 가쪽벽에 위·가운데·아래 3개의 코선반(비갑개, nasal concha)이 있고, 사이가 위·중간·아래콧길(비도)
  • 코선반은 소용돌이를 발생시켜 원심력에 의해 먼지를 코안 점막에 부딪치게 하여 먼지 제거
  • 코안 아래 입안의 앞쪽은 단단입천장(경구개, hard palate)으로 뼈가 받침, 뒤쪽의 물렁입천장(연구개, soft palate)에는 뼈가 없음

위기도 — 코곁굴 (Paranasal Sinuses)

  • 코곁굴(부비동, paranasal sinuses)은 코안 주위를 고리처럼 감싸는 공간
  • 구성: 이마굴, 나비굴, 벌집굴, 위턱굴
  • 나비굴(접형골동, sphenoid sinus): 코안 위쪽 나비뼈몸통 안, 위콧길 뒤쪽으로 개구
  • 벌집굴(사골동, ethmoid sinus): 벌집뼈 안 3~15개의 작은 굴, 위콧길과 가운데콧길에 개구
  • 이마굴(전두동, frontal sinus): 이마뼈 안 1쌍의 굴, 반달틈새를 통해 중간콧길에 개구
  • 위턱굴(maxillary sinus): 가장 큰 코곁굴, 위턱뼈몸통에 위치, 코곁굴염이 가장 잘 생김

위기도 — 인두 (Pharynx)

  • 인두(pharynx)는 소화관과 기도가 교차하는 부분, 길이 약 13cm, 깔때기 모양의 섬유근성 관
  • 코안·입안·후두의 뒤편, 목뼈 바로 앞에 위치
  • 공기는 코안 → 코인두(비인두, nasopharynx) → 입인두(구인두, oropharynx) → 후두인두(laryngopharynx) → 후두

아래기도 — 후두 (Larynx)

  • 후두(larynx)는 목소리 상자(voice box)로, 공기를 기관으로, 음식물은 식도로 보내고 발성 작용을 수행
  • 유리연골 8개 + 탄력연골(후두덮개) 1개로 구성, 넷째~여섯째목뼈 높이
  • 후두공간(후두강): 실주름과 성대주름에 의해 위·가운데·아래 3부로 구분
  • 성대문(성문, glottis): 좌우 성대주름 사이의 공간
  • 발성: 성대근육에 의해 성대문이 좁혀지고 공기가 진동하여 소리 발생
  • 실주름은 거짓성대(가성대)로 발성과 직접 관계없이 공기에 습기를 더함
  • 남녀차: 남성 성대 13mm, 여성 성대 10mm; 남성은 후두융기(Adam's apple) 돌출
  • 후두연골: 3개의 큰 연골(방패·반지·후두덮개연골) + 3쌍의 작은 연골(모뿔·쐐기·잔뿔연골)
    • 방패연골(갑상연골, thyroid cartilage): 후두연골 중 가장 큼, 방패 모양
    • 반지연골(윤상연골, cricoid cartilage): 반지 모양, 위로는 방패연골, 아래로는 첫째기관연골과 연결
    • 후두덮개연골(epiglottic cartilage): 테니스 라켓 모양, 삼킴 시 후두입구를 막아 음식물의 기도 유입 방지
    • 모뿔연골(피열연골, arytenoid cartilage): 피라미드 모양, 성대의 위치와 긴장도를 조절하여 발성에 관여

아래기도 — 기관 (Trachea)

  • 기관(trachea)은 후두 아래에서 기관갈림(기관분기부)까지, 직경 약 2.5cm, 길이 약 10~12cm
  • 일곱째목뼈에서 다섯째등뼈까지 내려감, 식도 앞을 수직 하행
  • 안쪽에 섬모 점막이 덮여 있어 먼지가 묻은 점액을 허파쪽에서 인두쪽으로 보냄
  • 15~20개의 C자형 유리연골고리로 구성
    • C자형 열린 쪽: 식도와 맞닿아 큰 음식 덩어리 삼킬 때 팽창 허용
    • 단단한 쪽: 기관을 받쳐주어 압력 변화에도 열려 있도록 유지

아래기도 — 기관지 (Bronchi)

  • 기관갈림에서 오른쪽·왼쪽 1차기관지(primary bronchus) 형성
  • 1차기관지: 다섯째등뼈 높이에서 기관이 두 갈래로 갈림. 오른쪽이 왼쪽보다 짧고 굵으며 수직에 가까움 → 이물질이 오른쪽으로 들어가기 쉬움
  • 2차기관지(엽기관지, lobar bronchi): 오른쪽 3개, 왼쪽 2개 분지 — 같은 쪽 허파엽 수만큼 분지
  • 3차기관지(구역기관지, segmental bronchi): 오른쪽 10개, 왼쪽 8개
  • 세기관지(bronchiole): 지름 1mm 이하
  • 종말세기관지(terminal bronchiole): 공기의 전도계, 한 소엽에 50~80개
  • 호흡세기관지(respiratory bronchiole): 길이 짧고 지름 약 0.5mm, 허파꽈리가 붙어 있어 가스교환 참여
  • 허파꽈리관(alveolar duct): 호흡세기관지에서 나오는 2~10개의 작은 관
  • 허파꽈리주머니(alveolar sac): 벽이 얇은 주머니 모양

허파 (Lung)

  • 허파(폐, lung)는 양쪽 가슴안(흉강)을 채우는 1쌍의 반원추형 실질기관
  • 용적: 오른쪽 1,200cc, 왼쪽 1,000cc; 중량: 오른쪽 600g, 왼쪽 500g (중량비 10:9)
  • 표면은 가슴막(흉막, pleura)에 싸여 있음
  • 허파꼭대기(폐첨, apex): 빗장뼈 위로 2~3cm 솟아 있음
  • 허파바닥(폐저, base): 가로막 위에 얹혀 있음
  • 허파문(hilus): 기관지, 허파동맥, 기관지 동·정맥, 림프관, 신경이 출입
  • 오른허파: 위·중간·아래엽의 3엽 (빗틈새 + 수평틈새)
  • 왼허파: 위·아래엽의 2엽 (빗틈새)

허파꽈리 (Alveoli)

  • 허파꽈리(alveoli)는 벽이 얇은 공기주머니, 수는 약 3~5억 개
  • 가스교환 표면적: 약 70~100㎡
  • 제1형 허파세포: 허파꽈리 면적의 95%, 얇고 표면적이 넓어 가스교환에 용이
  • 제2형 허파세포(사이막세포, septal cell): 표면활성제(surfactant) 분비, 날숨 시 허파꽈리가 달라붙는 것 방지
  • 먼지세포(dust cell): 허파꽈리의 큰포식세포(대식세포, alveolar macrophage), 포식작용

허파의 기능

  • ① 표면활성제(surfactant) 분비
  • ② 혈관확장 키닌(kinin) 분비
  • ③ 위호르몬 가스트린(gastrin) 분비
  • ④ 안지오텐신 I → 안지오텐신 II 전환
  • ⑤ 프로스타글란딘(prostaglandin) 합성·저장·방출, 인슐린 대사 관여 — 모두 허파모세혈관 내피에서 수행

세로칸 (Mediastinum)

  • 세로칸(종격, mediastinum)은 좌우 허파 사이의 가슴안 중앙부
  • 양쪽: 세로칸가슴막, 앞쪽: 가슴뼈, 뒤쪽: 척주, 아래쪽: 가로막
  • 위세로칸: 기관, 식도, 가슴샘, 대동맥, 위대정맥
  • 앞세로칸: 가슴샘, 속가슴동맥, 속가슴정맥
  • 중간세로칸: 심장
  • 뒤세로칸: 식도, 미주신경, 대동맥, 가슴림프관

2. 호흡 생리

호흡 기전 — 들숨운동 (Inspiration)

  • 들숨(흡식, inspiration)은 능동적 운동, 근육 수축으로 에너지 소비
  • 가로막 수축: 가로막신경(C₃~C₅) 흥분 → 가로막이 배안쪽으로 하강 → 가슴안 내압 -2.5~-6mmHg → 허파꽈리 확장, 용적 약 250~300ml 증가. 가로막 호흡 = 복식호흡(주로 남성)
  • 바깥갈비사이근 수축: 갈비뼈 상승 → 가슴우리 상승 → 가슴안 확대 → 내압 하강 → 허파꽈리 확장. 바깥갈비사이근 + 척주세움근 = 흉식호흡
  • 배근육 이완: 가로막 수축에 도움, 가로막 하강 보조
  • 허파꽈리 표면장력 약화: 표면활성제(surfactant, 지질단백질)가 표면장력 약화 → 허파꽈리 확장 용이

호흡 기전 — 날숨운동 (Expiration)

  • 날숨운동은 들숨의 반대작용으로 수동적 운동, 에너지 불필요
  • 주작용근: 속갈비사이근(내늑간근, internal intercostal muscle)

허파의 용적과 용량

  • 1회 호흡용적(TV, tidal volume): 안정 호흡 시 흡입/호출 공기량 = 500ml
  • 들숨 예비용적(IRV): 안정 시 들숨 후 더 들이마실 수 있는 양 = 3,100ml
  • 날숨 예비용적(ERV): 안정 시 날숨 후 더 내쉴 수 있는 양 = 1,200ml
  • 잔기용적(RV): 최대 날숨 후 허파에 남아 있는 양 = 1,200ml (총허파용량의 20%, 35% 이상이면 기능 이상)
  • 들숨용량(IC) = IRV + TV = 3,600ml
  • 기능적 잔기용량(FRC) = RV + ERV = 2,400ml
  • 허파활력량(폐활량, VC) = IC + ERV = 4,800ml (40세 이후 현저 감소)
  • 온허파용량(총폐용량, TLC) = VC + RV = 6,000ml
  • 무효공간(호흡사강, dead space): 허파꽈리에 도달하지 않는 약 150ml의 공기가 머무는 기도 부분

허파꽈리 안의 가스교환

  • 대기 조성: 질소 78.97%, 산소 21%, 탄산가스 0.03%
  • 기압 760mmHg 기준 분압: N₂ 600mmHg, O₂ 159mmHg, CO₂ 0.3mmHg
  • 허파꽈리 공기: O₂ 약 100mmHg, CO₂ 약 40mmHg, 수증기 47mmHg, N₂ 573mmHg
  • 동맥혈액: O₂ 약 95mmHg (일부 100), CO₂ 약 40mmHg
  • 정맥혈액(허파순환계): O₂ 약 40mmHg, CO₂ 약 46mmHg

혈액에 의한 산소 운반

  • 산소는 대부분 적혈구 내 헤모글로빈과 화학적 용해로 결합(옥시헤모글로빈), 물리적 용해는 약 1%
  • 산소포화도(oxygen saturation): 산소 분압이 높을수록 100%에 접근
  • 산소포화곡선(산소해리곡선): 탄산가스 분압이 높아지면 곡선이 오른쪽으로 이동 → 동일 O₂ 분압에서 CO₂가 높으면 산소포화도 낮아짐

혈액에 의한 탄산가스 운반

  • CO₂ 물리적 용해는 극소량, 대부분 화학적 용해
  • 약 80%: 중탄산이온(HCO₃⁻) 형태로 혈장 속에 용해
  • 약 20%: 헤모글로빈과 결합한 카바미노헤모글로빈(carbaminohemoglobin)

호흡운동의 신경성 조절

  • 호흡중추: 숨뇌(연수)~다리뇌(교뇌)에 분포
    • 숨뇌: 호흡주기 조절
    • 다리뇌: 호흡수 조절
  • 헤링-브로이어반사(Hering-Breuer reflex): 미주신경을 통한 호흡조절 반사, 호흡운동에 율동성 부여
  • 대뇌겉질은 자동적 기전에 우선하여 어느 정도 호흡 조절 가능 (예: 노래, 풍선 불기)

호흡의 화학적 조절

  • 혈중 CO₂가 호흡중추 흥분성에 영향
  • 말초화학수용체: 목동맥팽대(경동맥소체) 및 대동맥토리(대동맥소체) — CO₂ 농도↑ 또는 O₂ 농도↓ 감지 → 호흡중추 흥분성 증가 → 호흡 활발
  • 호흡중추 자체도 CO₂에 직접 흥분 → CO₂ 증가 시 호흡이 깊어지고 호흡수 증가

이상 호흡

  • 빠른호흡(빈호흡, tachypnea): 호흡빈도 증가
  • 호흡곤란(dyspnea): 호흡운동을 힘들게 하는 상태 — 기도 저항 증가(천식), 액체 저류(복막염·흉막염), 가스교환 장해(폐렴) 등
  • 체인-스토크스호흡(Cheyne-Stokes respiration): 약한 호흡/무호흡 후 점차 호흡이 커지고 다시 약해지는 패턴 — 중추신경 장애, 심장부전, 요독증, 마약중독, 임종 시
  • 쿠스마울호흡(Kussmaul respiration): 깊은 호흡이 느리게 계속 — 당뇨병, 콩팥부전 등 중증 산중독증(acidosis) 혼수 시
  • 비오호흡(Biot respiration): 무호흡/얕은 호흡 사이에 깊은 호흡이 불규칙하게 나타남 — 뇌압항진을 수반한 뇌염, 수막염, 뇌종양
